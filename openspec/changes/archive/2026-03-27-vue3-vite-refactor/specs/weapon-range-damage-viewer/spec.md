## MODIFIED Requirements

### 需求:入口与页面布局

系统必须在 **Vue 应用内**提供导航入口（例如首页或站点顶栏中的链接），使用户可进入武器距离伤害工具视图。武器工具必须为 **Vue 路由视图**（不得再依赖 `pages/` 下独立 HTML 文件作为唯一入口）。界面必须使用 **Reka UI** 实现表单与交互控件（替代 Pico CSS CDN）；允许项目级样式在 Reka UI 之上统一视觉。页面布局必须与地图工具视图同型：左侧为正方形地图视口（与地图工具相同的缩放、平移与测距能力），右侧为信息栏。右侧信息栏在水平方向上的占宽规则必须与地图工具视图一致，并满足 `map-range-viewer` 中「页面布局与地图视口尺寸」需求对**右栏占满除左侧地图以外剩余宽度**的约束。

#### 场景:从首页进入

- **当** 用户在应用内首页点击「武器距离伤害」或等价链接
- **那么** 必须进入武器工具路由且左侧地图与右侧栏可见

#### 场景:宽屏下右栏与地图页一致占满剩余宽度

- **当** 用户在宽屏桌面浏览器中打开武器工具视图且视口宽度足以并排显示左侧地图与右侧信息栏
- **那么** 右侧信息栏必须在水平方向上占满除左侧正方形地图以外的**全部**剩余可用空间，不得在信息栏右侧因栏宽上限留下大块空白

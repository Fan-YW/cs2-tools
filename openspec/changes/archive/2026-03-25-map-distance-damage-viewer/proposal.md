## 为什么

需要在浏览器中直观查看 CS 风格地图雷达图，并在图上测量与 C4 伤害衰减相关的距离与估算伤害，便于战术与数值理解。单页原生实现可降低依赖、便于部署在静态站点。

## 变更内容

- 新增一个使用 Pico CSS 的原生 HTML 页面（及必要的内联或同目录脚本/样式）。
- 左侧为正方形地图视口（高度贴近视口高度，宽度等于高度），从 **`public/png/`** 加载所选地图对应的 `*_radar_psd.png`；元数据从 **`public/json/`** 加载。
- 右侧为地图选择与距离/伤害信息区：下拉框列出 7 张地图；下方依次显示当前距离、基本伤害、计算伤害。**距离**在标注游戏单位的同时，**以括号**展示按 **1 unit = 25.4 mm** 换算的 **米**。
- 交互：滚轮在地图区域缩放；**左键按住拖动**平移地图；**单击**左键落点后在**不按左键移动**时更新测距圆（圆心为首次单击位置，半径至当前指针）；第二次左键固定圆并停止更新数值；右键清除圆与圆心。
- 地图显示须**始终铺满**正方形视口，缩放与平移后**不得**露出白边或未覆盖区域。
- 实现时先将各图 Valve 概览 `.txt` 转为结构化 JSON（同目录或约定路径），页面从 JSON 读取 `scale` 与 `c4_base_damage` 等字段，并按给定公式计算展示倍率与伤害。
- 首次左键落点后，在以落点为圆心处叠加参考圆：红色圆半径为 \(\sqrt{\ln(100/\text{baseDamage}) \cdot (-2\cdot7\cdot7/6/6)}\cdot\text{baseDamage}\)（游戏单位）；绿色圆半径为 **baseDamage × 3.5**（游戏单位）。**当测距距离大于 baseDamage×3.5 时，计算伤害显示为 0**；否则按指数衰减公式计算。

## 功能 (Capabilities)

### 新增功能

- `map-range-viewer`: 地图雷达展示、倍率显示、缩放、圆形测距与 C4 伤害估算，地图元数据（`scale`、`c4_base_damage`）的读取与展示，以及**游戏单位距离的米制括号展示**（1 unit = 25.4 mm）。
